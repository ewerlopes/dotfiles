* Programming

** Tools
*** Copilot

Let's install the unnoficial Copilot client for Emacs.

#+begin_src emacs-lisp
(use-package copilot
  :straight (:host github :repo "copilot-emacs/copilot.el" :files ("*.el"))
  :ensure t)
#+end_src

Now we configure the completion to be automatic.

#+begin_src emacs-lisp
(use-package copilot
  :straight (:host github :repo "copilot-emacs/copilot.el" :files ("*.el"))
  :ensure t)
#+end_src

Finaly, we configure the acceptance of suggestions.

#+begin_src emacs-lisp
(define-key copilot-completion-map (kbd "<tab>") 'copilot-accept-completion)
(define-key copilot-completion-map (kbd "TAB") 'copilot-accept-completion)
#+end_src
